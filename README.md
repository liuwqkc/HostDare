## 常见问题解答

**Q：优惠码可以叠加使用吗？**

不可以，每个订单只能用一个优惠码。但双倍资源福利是独立的，可以和优惠码同时享受。所以购买ASSD或CAMD系列时，先用优惠码打折，然后再申请双倍资源，相当于叠加了两项优惠。

**Q：CN2 GIA是双向还是单向？**

HostDare的CN2 GIA是回程优化，也就是从服务器到国内走CN2 GIA，去程是普通线路。如果需要双向CN2 GIA，可以考虑搬瓦工或DMIT，但价格会贵不少。

**Q：购买后可以换IP吗？**

前三天内免费换一次，超过三天每次换IP收费2美元。建议拿到IP后先测试一下，如果被墙或者延迟太高，赶紧在三天内申请更换。

**Q：支持IPv6吗？**

支持，但需要开工单向客服申请，会免费分配IPv6地址。如果你的应用需要IPv6，记得在开通服务后顺便申请一下。

**Q：可以安装Windows系统吗？**

可以，但有内存要求。一般来说2GB内存起步才能装Windows，低于这个配置装上去也跑不动。系统选项里提供了Windows 2003、2008、2012等版本。

**Q：续费价格会涨吗？**
